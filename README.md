# Ansioso 🌿

Aplicação web de apoio emocional e organização de pensamentos.

## Funcionalidades

- Diário Emocional
- Fato ou Interpretação
- Não Envie Agora
- Modo Crise
- Histórico de Registros
- Cadastro e Login
- Rede de Apoio
- Frases Motivacionais
- Contatos de Emergência

## Tecnologias

- Node.js
- Express
- EJS
- Sequelize
- MySQL
- JWT
- bcrypt

## Arquitetura

MVC (Model View Controller)

## Como executar

```bash
npm install
npm start
```

Acesse:

http://localhost:3000
