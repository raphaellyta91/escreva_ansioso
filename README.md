# Ansioso 🌿

Aplicação web desenvolvida com Node.js, Express, Sequelize e MySQL.

Projeto focado em apoio emocional e organização de pensamentos ansiosos.

## Funcionalidades

- Diário Emocional
- Não Envie Agora
- Fato ou Interpretação
- Modo Crise
- Histórico emocional
- CRUD completo
- Sequelize + MySQL
- MVC

## Tecnologias

- Node.js
- Express
- Sequelize
- MySQL
- EJS
- CSS

## Como executar

npm install

npm run dev
